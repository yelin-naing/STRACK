# Student Performance Tracking System

## Overview

The Student Performance Tracking System is a web-based application designed to help students monitor and improve their academic performance. The system provides tools for tracking grades, attendance, and learning progress while encouraging student engagement through gamification features such as badges, points, rewards, and leaderboards.

The project aims to support data-driven decision-making by providing students and lecturers with clear insights into academic performance and learning trends.

## Features

### Student Features

* User registration and login
* Secure authentication and password protection
* View academic grades and performance records
* Track attendance history
* Monitor learning progress through dashboards and charts
* Earn badges and rewards for achievements
* View points and leaderboard rankings

### Lecturer/Admin Features

* Manage student records
* Record and update grades
* Record attendance information
* Monitor student performance trends
* Generate reports and analytics
* Manage gamification elements

## System Objectives

The main objectives of the system are:

1. To provide students with a centralized platform for tracking academic performance.
2. To improve student engagement through gamification techniques.
3. To help lecturers monitor student progress effectively.
4. To support data-driven academic decisions through analytics and visualization.
5. To encourage continuous improvement in student learning outcomes.

## Gamification Features

The system includes several gamification elements:

* Achievement badges
* Reward points
* Leaderboards
* Progress tracking
* Performance milestones

These features are designed to increase motivation and encourage active participation in academic activities.

## Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/yelin-naing/STRACK.git
   ```

2. Navigate to the project folder:

   ```bash
   cd STRACK
   ```

3. Import the MySQL database.

4. Configure database connection settings.

5. Start the local server.

6. Open the application in your web browser.
