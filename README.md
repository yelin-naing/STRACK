**Student Performance Tracking System**

A web-based dashboard for tracking acdemic performance, with built-in gamification and privacy-first design. Developed as final-year project at Northumbria Univeristy. The system supports three user role: Students, Lecturers, and admins. This project aims to give students a clearer, more motivating view of their academic progress while keeping their data under their own control. 

**Features**
1. Role-based authentication for students, teachers, and admins
2. Academic tracking for grades, assignments, attendacne, and more.
3. Gamification for points, badges, and anomymous leaderboard
4. Role specific dashboards with tailored navigation per user role.
5. Notification alerts for announcements.
